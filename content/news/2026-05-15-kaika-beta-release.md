株式会社Seed は、 経営者 向け AI 右腕 サービス Kaika の ベータ版 を公開 しました。

## Kaika が できる こと

- **目標 の 整理**: 抱え 込み がち な 課題 を 会話 形式 で 分解 し 実行 可能 な 単位 に 落と し込む
- **議事録 の 整理**: 会議 の 記録 を 自動 で 構造 化、 決定 事項 と タスク を 抽出
- **右腕 agent の 実行**: 定 例 業務 を 自動 化 する 独 自 agent の 作成 と実行

## ベータ版 の 参加 方法

Kaika ベータ 参加 を ご希望 の 方 は、 お問い合わせ フォーム より ご連絡 ください。 先着 順 で ご 案 内 して います。

引き続き 改善 を進 めて まいります。
